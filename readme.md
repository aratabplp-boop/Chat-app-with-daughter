#　1.タイトル 一緒に勉強アプリ(This app is only used by Asako and Arata.)

## 2.課題内容（どんな作品か）
- 娘とリアルタイムで勉強時間を競い、励ましあえるアプリ
- 宣言メッセージを打ち込んで学習スタート
- スタートするとカウンターが動き出す
- スタートすると二人の好きなリラックマ写真がランダムに表示される（1分おきに変化）
- リアルタイム監視が走っており、相手が学習をスタートするとカウンター＆写真表示が動き出す
- いつでも応援メッセージを相手に送ることができる
- 学習をストップすると累積時間がカウンターに表示される
- ダッシュボードに学習履歴が蓄積される
- 娘（JK）との会話の中で、作ろうと思ったアプリ。世界中でたった二人だけのためのアプリです

- ダッシュボードにさりげなく「最後の学習から」の時間数カウントを忍び込ませる
